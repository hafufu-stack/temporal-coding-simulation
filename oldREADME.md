# Temporal Coding Simulation

## 背景

脳の神経細胞はスパイク情報が入る順序やタイミングを調節することで、扱える情報量を増やしているのではないかと修論で考察しました。
このリポジトリでは、それをSNN（Spiking Neural Network）で検証していきたいと思います。

因みに私の修論は海馬歯状回の顆粒細胞のシミュレーションでした。
この細胞には、空間情報を運ぶMD入力と非空間情報を運ぶLD入力があり、それらは相互作用があるようです。

- [ ] **次回**

    - 新たな暗号化や圧縮技術に応用
    - 実用的なライブラリとしてリリース

## プログラムの概要

### 10-neuron-memory
10個のニューロンを用い、最も効率良く情報を記憶させる方法を検証するため、独立符号化と相関符号化の2つの符号化方式で表現可能な情報量を比較・検証しました。

1.  **独立符号化** (`burst_phase_coding_10neurons.py`)
    - 各ニューロンが独立してバースト発火の「位相」と「ISI（発火間隔）」で情報を表現。
    - 時間分解能：dt = 0.1ms
    - 理論情報量：**約82ビット**

2.  **相関符号化** (`correlation_coding_10neurons.py`)
    - ニューロン間の「入力タイミングのズレ（相関）」を利用。
    - 1つのニューロンを基準クロック（MD役）とし、他（LD役）は相対時間で情報を表現。
    - 時間分解能：dt = 0.05ms
    - 理論情報量：**約111ビット** (情報量が大幅に向上)

修士論文における仮説「MD（内側貫通枝）とLD（外側貫通枝）の相互作用による情報量増大」を、Python上のLIF（Leaky Integrate-and-Fire）ニューロンモデルで再現しました。
本シミュレーションは、**「絶対的な発火時刻ではなく、MD入力を基準としたLD入力の相対的な遅延（位相差）こそが、高密度な情報の器である」** という仮説に基づいています。
脳の神経細胞（ニューロン）における「時間位相符号化（Temporal Phase Coding）」と「入力相関（Input Correlation）」の効果を検証するシミュレーションです。

## 動作環境

- Python 3.x
- NumPy
- Matplotlib
- Japanize-Matplotlib (日本語フォント表示用)

## 実行方法

必要なライブラリをインストール後、各スクリプトを実行してください。

```bash
pip install numpy matplotlib japanize-matplotlib
```

### 実験1：独立符号化モデル
```bash
cd 10-neuron-memory
python burst_phase_coding_10neurons.py
```

### 実験2：相関符号化モデル
```bash
cd 10-neuron-memory
python correlation_coding_10neurons.py
```

実行すると、シミュレーション結果の波形グラフが表示され、コンソールに達成した情報量（ビット数）が出力されます。


## 現時点での考察
**相関符号化** (`correlation_coding_10neurons.py`)の方がエントロピー（表現できる情報量）が上だったことから、ニューロンを1つ基準用に消費（犠牲に）してでも相対時間符号化を行う方が、システム全体のエントロピーが増大することが示唆されました。
